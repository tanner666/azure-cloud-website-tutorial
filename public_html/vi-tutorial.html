Vi Tutorial

What is vi?

Vi stands for Unix Visual Editor. It is a CLI (terminal) command that allows for the editing of files within the command line itself.

    vi is a full screen editor that comes with two modes of operation:

  1. Command Mode: Commands which cause action to be taken on the file.
  2. Insert Mode: Entered text is inserted into the file.

This tutorial will teach the very basics of vi. You will learn the basic commands and knowledge needed to read from, write to, and operate on files.

Syntax

  • vi <filename> opens a file in the editor.
    • If the file exists, then the first page of the file will be displayed. If the file does not exist, then an empty file and screen are created which can they be edited.

Editor Commands:

  • i: Enter insert mode. This is where you edit the contents of the file.
  • esc: Exit insert mode and enters command mode.
  • Colon (:): Moves the cursor to the command line when in command mode.
  • :x<Return> / :wq<Return>: Write the changes to the file (saves changes) and quit vi.
  • :w<Return>: Writes to the file (saves changes).
  • :q<Return>: Quit vi.
  • :q!<Return> Quits vi without saving changes.
  • :wq<Return>: Saves changes and quits vi.
